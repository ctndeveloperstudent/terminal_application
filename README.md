# Event Planning Terminal Application

## Features
- Main Menu
- Guest List Tracker
- To Do List 
- Expense Tracker

## Main App
- Main menu
- Options to select:
    - Guest list
    - To do list
    - Expense tracker
    - Exit App 

## Guest List Tracker
- Add guest
- View guest list
- Delete guest

## To Do List
- Add task
- View items
- Delete items

## Expense Tracker
- Add expense
- Add category (eg: guest, food, drink, venue, vendor, other)
- Add expected cost
- Add actual cost
- View expenses by category

## Dependencies
- Python 3.9
- Pyfiglet
- Colorama
- Tabulate (for expense tracker table)
